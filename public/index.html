Korean Dummy JSON API

Welcome to the Korean Dummy JSON API. Here are some available routes:

  • /users - User data
  • /posts - Posts data
  • /comments - Comments data
  • /todos - Todos data
  • /books - Books data
  • /reviews - Reviews data
  • /auth - Authentication
  • /image - Dynamic Image

For full documentation, visit: https://www.koreandummyjson.site/
