Korean Dummy JSON API Server

Korean Dummy JSON 프로젝트의 API Routes를 대신하여 Node.js로 구축한 API 서버입니다.

API Routes

  • /users - User data (20 users)
  • /posts - Posts data (100 posts)
  • /comments - Comments data (500 comments)
  • /todos - Todos data (200 todos)
  • /books - Books data (100 books)
  • /reviews - Reviews data (500 reviews)
  • /auth - Authentication
  • /image - Dynamic Dummy Image

자세한 내용은 GitHub Readme를 참고해주세요.
